durable::heap::retrieve

Looks up all recently accepted values higher than with proposal_id. 

virtual std::map< int64_t, std::string > retrieve(
    int64_t proposal_id);

This function does not necessarily need to return all values. In fact, it is preferred if a large catch-up is retrieved in small batches, so that a catch-up can occur gradually instead of in a single paxos round.
