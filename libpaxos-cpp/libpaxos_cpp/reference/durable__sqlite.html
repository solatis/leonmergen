Remarks

Requires the sqlite development headers to be installed, and libpaxos-cpp to be configured with enable-sqlite support

This class provides a durable storage backend based on sqlite. It opens a specific database file, handles input/output to that file and maintains a durable state. When a server is restarted using a previously used database file, it continues from the previous state.

Thread Safety

Distinct objects: Safe, as long as different database files are used

Shared objects: Unsafe

Examples

Set up a server that uses a sqlite database file called "db.sqlite". 

paxos::configuration configuration;
configuration.set_durable_storage (new paxos::durable::sqlite ("db.sqlite"));
paxos::server server ("127.0.0.1", 1337,
                      [] (int64_t proposal_id, std::string const & input) -> std::string
                      {
                          return input;
                      },
                      configuration);
