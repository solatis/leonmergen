configuration::set_majority_factor

Controls the percentage of servers that must be reachable before progress is made. 

void set_majority_factor(
    double factor);
Parameters

factor

The factor of servers that need to be alive

0.0 < factor && factor <= 1.0

Defaults to 0.5
