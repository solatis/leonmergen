durable::sqlite::store

Stores an accepted value. 

virtual void store(
    int64_t proposal_id,
    std::string const & byte_array);
Parameters

proposal_id

The id of the proposal to store

byte_array

The value that is associated with the proposal

Preconditions

highest_proposal_id () == (proposal_id - 1)

Postconditions

highest_proposal_id () == proposal_id
