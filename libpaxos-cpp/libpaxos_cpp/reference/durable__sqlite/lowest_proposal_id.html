durable::sqlite::lowest_proposal_id

Looks up the lowest proposal id currently stored. 

virtual int64_t lowest_proposal_id();
Return Value

Returns the lowest proposal id in history, or 0 if no previous proposals are stored

This functionality is not really required by the paxos library, but is used by the test cases.
