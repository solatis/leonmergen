server::wait

Blocks until internal worker thread has stoppped. 

void wait();

This function blocks until the internal worker thread stops, which should never occur. It is useful to let the main () thread block forever.

Remarks

This function will return immediately when an external worker thread is used to control the boost::asio::io_service object
